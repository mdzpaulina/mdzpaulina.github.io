Preguntas HTML + CSS

HTML

Introducción

  1. ¿Qué es HTML?

    HTML es un lenguaje que se utiliza para crear páginas web, dandole estructura mediante etiquetas.

  2. Explica las etiquetas de head y body

    La etiqueta head contiene la información sobre la página que no se muestra en el contenido de ella, como pueden ser metadatos, o el titulo de la pagina que podemos ver en el navegador, o enlaces a archivos CSS, o scripts de JavaScript.
    El body contiene el contenido principal de la página web, que incluyen encabezados o párrafos, imagenes, enlaces, listas o tablas, etcétera.

Basic Tags

  1. Explica las etiquetas section, article, aside, header, footer

    La etiqueta section se usa para agrupar contenido en la página. Por ejemplo se puede usar para dividir la página por temas o categorias.
    La etiqueta article se usa para seccionar contenido que puede existir por si solo.
    La etiqueta aside se utiliza para contenido de la pagina que no es esencial, como por ejemplo alguna publicidad.
    El header es la cabecera de una página, que usualmente incluye el logo, el menú de navegación y el título.
    El footer representa el pie de página que usualmente contiene información de contacto y derechos de autor.

CSS

Introducción

  1. ¿Qué es CSS?

    CSS es un lenguaje que se utiliza para dar un estilo o dar cierta apariencia a una página web. Nos permite personalizar más cada elemento de nuestras páginas.

  2. ¿Cuál es el propósito de CSS?

    El propósito de CSS, como lo había mencionado antes, es podernos permitir personalizar cada elemento y etiqueta de nuestra página, y nos permite también que la página se adapte para distintos dispositivos.

  3. ¿Cómo aplicas CSS a un HTML?

    Para poder aplicar CSS a un documento HTML, hay 3 formas para hacerlo.

    1. Mediante un archivo de CSS externo, se crea un archivo css separado del archivo html y se enlaza en el head del archivo html usando link
    2. Mediante la etiqueta style dentro del head, en el mismo archivo html en que se esta trabajando.
    3. Directamente en las etiquetas que se quieran modificar, mediante el atributo style, es util para ajustes rapidos

  4. ¿Qué es un selector? ¿Qué es una regla de CSS?

    Un selector en CSS se usa para identificar uno o varios elementos de HTML y aplicarles estilos. Una regla de CSS es el conjunto de estilos que se asignan a los elementos seleccionados. Cada regla de CSS tiene dos partes principales, el selector que define a qué elementos de HTML se les aplicarán los estilos y las declaraciones que son las propiedades y valores que determinan cómo se verán esos elementos.

CSS Básico

  1. Explica las Tag Selector, Class Selector, Tag + Class Selector, Id Selector
    • Los Tag Selector aplican estilos a todos los elementos de una eiqueta HTML específica.
    • Los Class Selector aplican estilos a cualquier elemento que tenga una clase especifica.
    • El ID Selector aplican estilos a un único elemento con un ID especifico.
    • Cuando se usa combinado el tag + class selector, especifica que solo las etiquetas que tengan una clase particular tendrán ese estilo.

  2. ¿Qué es una clase? ¿Qué es un id? ¿Cómo estan relacionadas a CSS?
    • Una clase se utiliza para aplicar estilos a múltiples elementos. Un mismo nombre puede asignarse a diferentes etiquetas para así poder reutilizar estilos y evitar repetir codigo.
    • Un ID es único en la página y solo puede asignarse a un elemento.
    • Ambos estan relacionados ya que se usan para asignar estilos, pero la gran diferencia entre ellos es que una clase puede reutilizarse en múltiples elementos mientras que un ID se puede utilizar en un solo elemento.

  3. Explica el CSS Box Model, muestra un ejemplo

    El CSS Box Model es una representacion de la estructura de cada elemento CSS en una pagina. Se podria decir que cada elemento se representa y se comporta como una caja. Cada caja tiene un content, un padding, un border y un margin.

  4. ¿Cómo puedes cambiar el color del font usando CSS?

    Para cambiar el color del font, se puede usar la propiedad color, en esta propiedad se pueden usar colores que ya vienen predefinidos, codigos hexadecimales, RGB y HSL.
